Note on notation:

We use the meta-symbol ≡ as a shorthand for "is logically equivalent to". It's a meta-symbol because it's not a symbol in sentence logic, but one that we use to talk about sentence logic. Unfortunately, Teller uses the same symbol for biconditionals, which we will learn about in the next module. I will have to introduce a different symbol for that.

Activity: Do this handout, and then play around with the Randomized Wason Selection Task

Note on notation:

Conditionals: For this class we will use → for conditionals. Teller uses another often used symbol ⊃, but sometimes that confuses people because that looks exactly like the symbol for superset in set theory, so I like → better. Nevertheless, there is indeed some deep connection between the conditional relationship in logic and superset in set theory - this will be a good presentation topic.

Biconditionals: For biconditional Teller uses the symbol ≡, which we have reserved for denoting logical equivalence, so instead we will use the symbol ↔. I think this is a better symbol anyway, for it represents nicely the idea that a biconditional statement is just a symmetrical conditional statement.
